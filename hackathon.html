The Challenge

Millions reconnect with the Quran during Ramadan — but many struggle to maintain that connection afterwards. This hackathon challenges you to build solutions that make that relationship lasting.

Improve accessibility, deepen understanding, build habits, or introduce entirely new ways to engage with the Quran using the Quran Foundation's powerful API ecosystem.

Technical Requirements

Your project must use at least one API from each category

Content API

Use at least one Quran Foundation Content API (or Quran MCP):

  • Quran APIs
  • Audio APIs
  • Tafsir APIs
  • Translation APIs
  • Post APIs (Lessons & Reflections)

User API

Use at least one Quran Foundation User API:

  • Bookmarks
  • Collections
  • Streak Tracking
  • Post APIs (Post a reflection)
  • Activity & Goals APIs

Judging Process

Phase 1 — Eligibility Screening: Submissions are checked against technical and submission requirements.

Phase 2 — Evaluation: Qualified submissions are independently scored by judges from Quran Foundation, Provision Launch, and technology/design professionals.
